# Titanic EDA Project

## Project Overview
This project performs Data Cleaning and Exploratory Data Analysis (EDA) on the Titanic dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Steps
- Data Cleaning
- Handling Missing Values
- Data Visualization
- Finding Survival Patterns

## Conclusion
Women and passengers in higher class had higher survival rate.
